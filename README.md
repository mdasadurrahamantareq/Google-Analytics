# Google-Analytics
# GA4 Learning Hub

> **Understand Google Analytics 4 — Simply.**

A fully self-contained, interactive learning hub for Google Analytics 4 (GA4). No build tools, no dependencies, no backend — just open `index.html` in a browser.

## 🔗 Live Demo

[View on GitHub Pages](https://mdasadurrahamantareq.github.io/Google-Analytics/)

---

## ✨ Features

- 📚 **Core Concepts** — Key GA4 terms explained with card-based UI
- 🧮 **Metric Calculators** — Bounce rate, session duration, conversion rate & more
- 🔽 **Funnel Visualiser** — Interactive acquisition → purchase funnel with drop-off analysis
- 📖 **Glossary** — Searchable A–Z reference of GA4 terms
- 🧠 **Quiz** — 10-question knowledge check with scoring
- 🗺️ **Reports Map** — Visual overview of every GA4 report section

## 🚀 How to Publish on GitHub Pages

1. **Create a new GitHub repository** (e.g. `ga4-learning-hub`)
2. **Upload these files** to the repository root:
   - `index.html`
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose **branch:** `main` and **folder:** `/ (root)`
6. Click **Save** — your site will be live at:
   ```
   https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
   ```
